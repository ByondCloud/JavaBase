## java中有7中位运算

### 一、按位与 &

#### 两位全1为1

```java
    10011
   +11011
  ——————————
    10011
```

### 二、按位或 |

#### 两位有1位为1，则为1，全0为0

```java
    10101
   +11001
 ————————————
    11101
```

### 三、按位异或 ^

#### 两位不同为1，两位相同为0

```java
    11001
   +10001
 ——————————
    01000
```

### 四、按位取反 ~

#### 0变1,1变0

注意：所有运算都需要转换到补码进行运算，所以我们先要从原码转换成补码

```java
十进制: -2
原码: 10000010
反码: 11111101
补码: 11111110
因为首位为0，三码合一，所以不用再转成原码了
```

### 五、算数右移 >>

本质就是除以多少次2

```java
int a = 184 >> 2
补码: 10111000    184
右移: 00101110    46
本质: (184/2)/2 = 46
```

### 六、算术左移

本质就是乘多少次2

```java
int a = 5 << 2
补码: 00000101    5
左移: 00010100    20
本质: 5 * 2 * 2 = 20
```
